About this Tool

This web tool was designed with the intent to provide simple chord numeral and name identification for a sequence of chords organized as a progression. Chords can be provided for the progression as a sequence of notes in each available chord field in the Progression Builder form, up to a maximum of 20 chords.

Logic is in place to identify chords of the following qualities:

  • Major, minor, augmented, and diminished triads
  • Suspended chords (sus2 and sus4)
  • Flat 5 chords and add5 chords
  • Major and minor seventh chords
  • Half-diminished and fully-diminished seventh chords
  • Dominant 7 chords

Note: Chords with added extensions such as 9ths, 11ths, and 13ths are not supported and may not be identified as expected.

Using the Form

The Chord Progression Builder Form has a few options available for generating a visual chord progression, described below.

Key: This determines the key for the chord sequence or progression for numeral analysis.

Time: This sets the time signature for the chord progression, and is purely a visual setting.

Chord Display Format: This determines how the chords are grouped and how each note's beam will behave.

Piano: This option will draw and group notes C4 and lower within the bass stave, and will draw and group those above in the treble stave.

SATB: This option expects four notes representing each voice in classical four-part harmony writing. For more information about this form of notation, see the SATB Info page.

Analyze Option: This option determines whether the chords provided will be assessed according to SATB part-writing rules.

** For the full list of rules included in this analysis, see the aforementioned SATB Info page.

Entering Chords

Chords are entered to the form in comma-separated format by specifying the name of the note and the octave it belongs to.

The accepted note names range from A - F with support for accidentals. The accepted note octave value ranges from 0 - 8, though notes at extremely low or high octaves may not be easily visible when drawn.

Notes can be specified with the accidentals: 'bb', 'b', '#', and 'x' representing a double flat, single flat, single sharp, and double sharp respectively.
